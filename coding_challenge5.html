Coding Challenge 5

John and his family went on a holiday to 5 different restruants 

        Bills

        1. $124
        2. $48
        3. $268
        4. $180
        5. $50

John likes to tip in the following way

  • If bill < $50 , John tips 20% of the bill
  • If $ 50 < bill < $200 , John tips 15% of the bill
  • If bill > $200 , John tips 10% of the bill

Implement a tip calculator using objects and loops:

  1. Create an object with array for the bill values.
  2. Add a method to calculate the tip.
  3. This method should include a loop to iterate over all paid bills and do tip calculation.
  4. As an output, create
    • a new array containing all the tips.
    • array containing final paid amounts (bill + tip).

Hint: Start with two empty arrays and fill them up in the loop.


Mark's family also went on a holiday going to 4 different restruants. The bills were: $77, $375, $110 and $45.

Mark likes to tip in the following way

  • If bill < $100 , Mark tips 20% of the bill.
  • If $ 100 < bill < $300 , Mark tips 10% of the bill.
  • If bill > $300 , Mark tips 25% of the bill.
  1. Implement the same functionality as before using Mark's tipping rules.
  2. Create a function (not a method) to calculate the average of a given array of tips.
    • HINT: Loop over the array and in each iteration store current sum in a variable(starting from 0). After you have the sum of the array, divide it by the number of elements in it (average)
  3. Calculate the average tip for each family.
  4. Log to the console which family paid the highest tips on average.
