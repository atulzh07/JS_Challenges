Coding Challenge 1

        Mark and John are trying to calculate their BMI
        BMI = mass/height^2 = mass/(height*height)
        mass should be in kg and height should be in metres 

        1. Store Mark's and John's mass and height in variables
        2. Calculate both their BMIs
        3. Create a boolean variable containing information whether 
           Mark has a higher BMI than John
        4. Print a string to the console containing the variable 
           from Step 3
